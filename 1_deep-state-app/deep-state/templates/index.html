DEEP STATE

Social Media Sentiment Analysis using Twitter

Presidential Impeachment

SENTIMENT

Discussion of impeachment of the U.S. president is a serious topic and even just the discussion alone, as well as the actual act of presidential impeachment, has a material impact on the daily lives of U.S. citizens and denizens of the globe. Having a solid understanding of the situation helps to inform individuals and organizations on how they should best adapt to any fallout and new potential presidential regime. The Deep State App is a tool to help provide light to this understanding.

The Deep State App is a social media sentiment analysis with the purpose to assess the current sentiment on presidential impeachment; basically, how do people feel about the impeachment of the U.S. president. The social media platform chosen to assess current sentiments is Twitter because Twitter provides live accessible textual information/posts from a broad array of the U.S./global population. The Deep State App uses the Vader Sentiment Analysis python code to assess each tweet's sentiment and determine a sentiment score. This data and additional metadata is then stored in a postgresql database. The Deep State App then visualizes this data.
